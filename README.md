# Costera N1 - Luxury Vacation Rental Landing

Plataforma web de alto rendimiento diseñada para la promoción y captación de clientes para un alojamiento turístico premium en el **Casco Antiguo de Villajoyosa**.

El proyecto funciona como un **embudo de conversión (SEO Funnel)**, optimizando la visibilidad orgánica para redirigir tráfico cualificado hacia el portal de reservas oficial gestionado por la inmobiliaria.

##  Stack Tecnológico

Este proyecto utiliza las versiones más recientes del ecosistema React para garantizar el mejor rendimiento, SEO y mantenibilidad:

* **Framework:** [Next.js 16 (App Router)](https://nextjs.org/) - Para renderizado eficiente y optimización de rutas.
* **Core:** [React 19](https://react.dev/) - Aprovechando las últimas mejoras en gestión de estados y hooks.
* **Estilos:** [Tailwind CSS 4](https://tailwindcss.com/) - Utilizando el nuevo motor de alto rendimiento para estilos ultra-ligeros.
* **Animaciones:** [Framer Motion](https://www.framer.com/motion/) - Para una experiencia de usuario fluida y de alta calidad visual.
* **Iconografía:** FontAwesome (SVG Core).

##  Características Principales

* **SEO Técnico:** Implementación de metadatos dinámicos y arquitectura semántica para mejorar el posicionamiento en buscadores.
* **Performance:** Optimización de imágenes (WebP/Lazy loading) mediante el componente `next/image` para asegurar tiempos de carga mínimos.
* **UI/UX Premium:** Animaciones coreografiadas con Framer Motion para reflejar la exclusividad del alojamiento.
* **Responsive Adaptive Design:** Adaptación total a dispositivos móviles para facilitar la consulta rápida de usuarios en movilidad.
* **Custom Hooks:** Implementación de lógica propia para la gestión de estados de interfaz y breakpoints dinámicos.
